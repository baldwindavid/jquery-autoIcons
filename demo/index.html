OPTIONS

Alternate image path

By default, icons are expected to be in the same directory as the displayed page. This can be changed via the prependPath option.

The icon for this Excel File is in an alternate directory.

Alternate class automatically applied to matching elements

By default, the icon class is automatically applied to any matching elements. This can be changed via the addClass option.

This Word File has a different class automatically applied. Obviously a css rule for "alternate-class" is needed.

CUSTOM MAPPING

The autoIcon plugin already includes a few common mappings (pdf, xls, ppt, etc.), but you can easily add to or change this list to your liking. Here is an example that maps the .txt extension to an image named txt.gif...

$.fn.autoIcons.mapping['a[href$=".txt"]'] = 'txt.gif'
